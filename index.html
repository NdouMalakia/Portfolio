How I Built My Portfolio Website

A breakdown of how I designed and developed my portfolio using HTML, CSS, and JavaScript.

🎨 Planning the Design

Before writing any code, I first planned the layout structure of my website. I decided to create sections like Home, About, Projects, and Blogs. I wanted a modern, dark-themed interface with smooth animations.

🧱 Structuring with HTML

I used semantic HTML to organize the content properly. Each section was wrapped inside clear sections, making it easier to style and maintain.

🎨 Styling with CSS

I implemented a dark color scheme using navy and black tones. Glassmorphism effects, smooth hover animations, and responsive design techniques made the website feel modern.

⚡ Adding Interactivity with JavaScript

JavaScript was used to add smooth scrolling, toggle buttons, and interactive blog sections. This improved user experience and made the site feel dynamic.This was a bit challenging for me.

🎯 What I Learned

Building this portfolio improved my frontend skills, strengthened my understanding of responsive design, and gave me confidence to build more advanced projects.
